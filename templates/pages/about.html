О проекте

DjangoArchitectAPI — это профессиональная демонстрация архитектуры Django REST API корпоративного уровня с оптимизациями и best practices.

Технологический стек

  • Backend: Django 4.2, Python 3.11
  • Database: PostgreSQL 15
  • Cache: Redis 7
  • Deployment: Docker, Docker Compose
  • Server: Gunicorn

Основные возможности

  • Публикация и управление постами
  • Категоризация контента
  • Географические метки (локации)
  • Отложенная публикация
  • Оптимизированные запросы к БД
  • Кэширование для высокой производительности
  • Пагинация
  • Адаптивный дизайн

Архитектура

Проект демонстрирует применение best practices:

  • Class-Based Views (CBV) для чистой архитектуры
  • Custom Model Managers для оптимизации запросов
  • Select/Prefetch Related для предотвращения N+1 проблемы
  • Redis для кэширования и сессий
  • Database Indexes для производительности
  • Разделение настроек на dev/prod окружения
  • Докеризация всего стека

Цель проекта: Демонстрация профессиональных навыков в разработке на Django и готовность к production-окружению.
